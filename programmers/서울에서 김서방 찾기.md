# 서울에서 김서방 찾기


<br>

## 풀이
```
class Solution {
    public String solution(String[] seoul) {
        int answer = 0;
        
        for (int i = 0; i < seoul.length; i++) {
            if ("Kim".equals(seoul[i])) {
                answer = i;
                break;
            }
        }
        return "김서방은 " + answer + "에 있다";
    }
}
```
